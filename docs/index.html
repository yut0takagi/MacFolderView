MacFolderView

メニューバーからワンクリック。
軽量なファイルブラウザ for macOS

ダウンロード（macOS）

v1.0.0 · macOS 14+ · Apple Silicon

⚡

メニューバー常駐

Dockに表示されず、メニューバーからすぐアクセス。作業の邪魔になりません。

📁

サイドバー

ホーム・デスクトップ・書類など、よく使うフォルダにワンクリックで移動。

🔍

検索とソート

名前・日付・サイズ・種類でソート。リアルタイム検索で素早くファイルを発見。

⌨️

キーボード操作

矢印キーで移動、Enterで開く、スペースでQuick Look。マウスなしで操作可能。

インストール方法

1

ダウンロード

上のボタンから MacFolderView.zip をダウンロードします。

2

解凍して移動

ZIPを解凍し、MacFolderView.app を「アプリケーション」フォルダにドラッグ&ドロップします。

3

初回起動

アプリを右クリック →「開く」を選択してください（下記の注意事項を参照）。

4

完了

メニューバーにフォルダアイコンが表示されます。クリックして使い始めましょう。

⚠ 初回起動時の「開発元が未確認」警告について

このアプリはAppleの署名がないため、初回起動時に警告が表示されます。以下の手順で開いてください：

  1. MacFolderView.app を右クリック（またはControlキー+クリック）
  2. メニューから「開く」を選択
  3. 確認ダイアログが表示されたら、もう一度「開く」をクリック

※ 2回目以降は通常どおりダブルクリックで起動できます。

動作環境：macOS 14 (Sonoma) 以降 · Apple Silicon (M1/M2/M3/M4)
